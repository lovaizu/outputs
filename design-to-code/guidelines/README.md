# Coding Guidelines

Guidelines for HTML / CSS / JS when coding from design (LP template based).

- [HTML](html.md) — document setup, semantics, naming, responsive images, BudouX
- [CSS](css.md) — file structure, breakpoint, color palette, layout utilities, BEM naming
- [JS](js.md) — stack, file layout, scroll, animations, sticky button, forms

## Reference material

`input/` contains the LP template that these guidelines were derived from:

- `input/site/index.html` — full component inventory
- `input/site/style.css` — base styles and all component CSS
- `input/site/js/function.js` — custom behavior
