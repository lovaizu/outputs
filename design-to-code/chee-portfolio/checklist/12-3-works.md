### C1 原因分析

`.work-card-title a` に `flex: 1` が指定されているが、`.work-card-arrow-link` も `<a>` なのでこのルールが適用され `flex-grow: 1` を持つ。結果、タイトルリンクと矢印リンクが空間を等分し、タイトルが約370pxしか確保されず折り返しが発生。

**修正案:** `.work-card-arrow-link` に `flex: 0 0 auto` を追加して矢印リンクの flex-grow を打ち消す。

### C2 原因分析

テンプレート `sec02-works.php` が `category_label` メタフィールドを括弧内に表示しているが、カンプでは `post_title`（例: "眉毛アートメイクLP"）が括弧内に表示されている。

**修正案:** テンプレートの括弧ラベルを `$work->post_title` に変更する。
